## 🎭 영상, 음악 플레이어 sunshine40-webplay
** 영상, 음악 연습용 웹 기반 플레이어입니다. 
이 프로젝트는 공연 준비에 필요한 다양한 오디오, 비디오, PDF 파일을 웹에서 손쉽게 다운로드하고, 재생할 수 있도록 설계되었습니다. 
GitHub Pages에서 호스팅되며, 모든 파일은 안전하게 다운로드 및 스트리밍이 가능합니다. 

## 📦 주요 기능
- **오디오 및 비디오 파일 재생**: 다양한 오디오(WAV)와 비디오(MP4) 파일을 선택하여 웹에서 바로 재생 가능합니다.
- **PDF 파일 다운로드**: PDF 파일을 쉽게 다운로드할 수 있도록 지원합니다.
- **GitHub에서 파일 다운로드**: GitHub에서 파일을 직접 다운로드할 수 있는 기능 제공.
- **모바일 및 데스크탑 반응형**: 모바일 및 데스크탑에서 모두 지원되는 반응형 웹 플레이어.

## 📈 사용 방법

[**sunshine40-webplay 바로가기**](https://jongcheon-kim.github.io/sunshine40-webplay/)  
위 링크를 클릭하여 **sunshine40-webplay** 웹사이트에 바로 접속하고, 파일을 다운로드하거나 재생할 수 있습니다.
웹사이트에 접속하면, 다양한 오디오, 비디오 및 PDF 파일을 선택할 수 있는 리스트가 표시됩니다. 원하는 파일을 선택하여 다운로드하거나 바로 재생 가능합니다.
