# sunshine40-webplay

**sunshine40-webplay**는 다양한 미디어 파일을 웹브라우저에서 선택하고 다운로드할 수 있도록 하는 웹 애플리케이션입니다. 이 프로젝트는 GitHub Pages에서 호스팅되며, 오디오, 비디오, PDF 파일 등을 선택하고 재생할 수 있는 기능을 제공합니다.

## 📦 주요 기능
- **오디오 및 비디오 파일 재생**: 다양한 오디오(WAV)와 비디오(MP4) 파일을 선택하여 웹에서 바로 재생 가능합니다.
- **PDF 파일 다운로드**: PDF 파일을 쉽게 다운로드할 수 있도록 지원합니다.
- **GitHub에서 파일 다운로드**: GitHub에서 파일을 직접 다운로드할 수 있는 기능 제공.
- **모바일 및 데스크탑 반응형**: 모바일 및 데스크탑에서 모두 지원되는 반응형 웹 플레이어.
